Project SHANKS.

## Structure

- `docs/`: Documentation (DAA, DAT, DCIP, OpenAPI, ADR, Diagrams)
- `contracts/`: Shared contracts and fixtures
- `frontend/`: ISM Portal (Next.js)
- `backend/`: ISM API (FastAPI)
- `ingestion/`: Ingestion Engine (Batches)
- `infra/`: Infrastructure configuration (Docker, CI)
- `scripts/`: Development scripts
